### Month 2: JavaScript + Git

#### JavaScript (3 weeks)

Week 1: Syntax

- Variables: let, const (don't learn var)
- Types: string, number, boolean, array, object, null, undefined
- Operators, conditions (if/else, switch), loops (for, while)
- Functions: function declaration, arrow functions
- Array methods: push, pop, map, filter, forEach

Week 2: DOM

- querySelector, querySelectorAll
- addEventListener (click, input, submit)
- classList (add, remove, toggle)
- textContent, innerHTML, value
- createElement, append (if needed to add dynamically)

Week 3: Asynchronous

- fetch (GET, POST)
- async/await
- .then/.catch (basics)
- JSON.parse, JSON.stringify
- LocalStorage (setItem, getItem)
